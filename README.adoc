== How to set up 
* Get the Emacs text-editor
** Link: https://www.gnu.org/savannah-checkouts/gnu/emacs/emacs.html
** Open up the editor
** `ALT + X`
*** `package-install-file`
*** Install the theme: `{path}/emacs-cotton-candy.el`
** `ALT + X`
*** `load-theme`
**** `emacs-cotton-candy`
